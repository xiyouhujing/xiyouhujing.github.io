Java笔记：this关键字

发表于 2019-06-05 | 分类于 Java | 阅读次数: ℃
字数统计: 1.6k | 阅读时长 ≈ 6

this是Java中的关键字，除了可以用来强调调用的是本类中的方法，this还有以下作用：

  1. 表示类中的属性
  2. 可以使用this调用本类的构造方法
  3. this表示当前对象
